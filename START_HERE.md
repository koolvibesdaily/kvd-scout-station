## 🆘 Stuck? Common Questions

**Q: "I see red X on the workflow"**
- A: Click the red X job to see what failed
- Common issue: Repo still private
- Solution: Make repo public first (Step 1)

**Q: "I completed all steps but site shows 404"**
- A: Wait 5 more minutes, DNS takes time
- Clear your browser cache (Ctrl+Shift+Delete)
- Try a different browser

**Q: "Where's the build job?"**
- A: Make sure repo is PUBLIC first
- Workflow won't run on private repos
- After making public, manually trigger again

**Q: "I don't see 'Deploy from a branch' option"**
- A: Make repo public first
- Then go back to Settings → Pages
- Option should appear

**Q: "How long does deployment take?"**
- A: Total time: ~4 minutes
  - Build: 2-3 minutes
  - Deploy: 1 minute
  - DNS propagation: ~1 minute
